# Assignment 3: Sales Data Analysis Using SQL

## Project Overview

This project analyzes the Superstore Sales dataset using SQL. The main objective is to apply advanced SQL concepts such as Subqueries, Common Table Expressions (CTEs), Window Functions, and JOIN operations to answer business-related questions and generate meaningful insights from sales data.

---

## Objective

* Analyze sales data using SQL.
* Apply Subqueries to solve filtering and comparison problems.
* Use CTEs for aggregations and reusable query logic.
* Use Window Functions for ranking and row numbering.
* Generate customer sales insights through business queries.
* Create a final report combining customer details, total sales, and rankings.

---

## Step 2: SQL Analysis

### Query 1: Orders Above Average Sales

**Purpose:** Find orders whose sales are greater than the average sales value.

**SQL Concept Used:** Subquery

**Insight:** High-value transactions contribute significantly to overall revenue.

---

### Query 2: Highest Order Per Customer

**Purpose:** Identify the highest sales order placed by each customer.

**SQL Concept Used:** Correlated Subquery

**Insight:** Helps understand customers' maximum purchasing behavior.

---

### Query 3: Total Sales Per Customer

**Purpose:** Calculate total sales generated by each customer.

**SQL Concept Used:** CTE

**Insight:** Identifies customers contributing the most revenue.

---

### Query 4: Customers with Above-Average Sales

**Purpose:** Find customers whose total sales exceed the average customer sales.

**SQL Concept Used:** CTE + Subquery

**Insight:** Highlights high-performing customers.

---

### Query 5: Customer Ranking

**Purpose:** Rank customers according to total sales.

**SQL Concept Used:** Window Function (RANK)

**Insight:** Helps identify top-performing customers.

---

### Query 6: Row Number for Orders

**Purpose:** Assign row numbers to orders within each customer.

**SQL Concept Used:** ROW_NUMBER() with PARTITION BY

**Insight:** Useful for analyzing customer purchase history.

---

### Query 7: Top 3 Customers

**Purpose:** Identify the top three customers based on total sales.

**SQL Concept Used:** Window Function

**Insight:** Reveals the highest revenue-generating customers.
