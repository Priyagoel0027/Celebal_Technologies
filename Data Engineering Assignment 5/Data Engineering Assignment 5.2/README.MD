# Week 5 – Apache Spark (PySpark) Assignment

## 📌 Overview

This project demonstrates the use of **Apache Spark (PySpark)** for performing data processing, transformation, and analytical operations on structured datasets. It provides both theoretical understanding and practical implementation of Spark DataFrame APIs, enabling efficient handling of large-scale data.

The assignment focuses on core big data concepts such as data cleaning, filtering, aggregation, schema management, and distributed data processing using Spark.

---

## 🎯 Objectives

- Understand limitations of traditional MapReduce architecture  
- Learn Spark In-Memory Computing and its advantages  
- Perform data cleaning and preprocessing using PySpark  
- Apply filtering and conditional transformations on datasets  
- Utilize `groupBy()` and aggregation functions for analysis  
- Handle missing values using `.na.drop()` and `.na.fill()`  
- Understand Spark DataFrame immutability  
- Perform schema transformation and data type casting  
- Explore shuffle operations and wide transformations  
- Build end-to-end data processing pipelines using PySpark
